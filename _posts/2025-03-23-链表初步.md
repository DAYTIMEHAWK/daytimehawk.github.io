#### 链表示例：
```C++
private:
    template <typename E>
    class Node {
    public:
        E val;
        Node* next;
        Node* prev;

        Node(Node* prev, E element, Node* next) {
            this->val = element;
            this->next = next;
            this->prev = prev;
        }
    };
```

```Python
class Node:
    def __init__(self, prev, element, next):
        self.prev = prev
        self.val = element
        self.next = next
```
